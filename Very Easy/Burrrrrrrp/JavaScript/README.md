Burrrrrrrp

Create a function that returns Burp with num "r"s in it.

Examples
longBurp(3) ➞ "Burrrp"

longBurp(5) ➞ "Burrrrrp"

longBurp(9) ➞ "Burrrrrrrrrp"

Notes
Expect num to always be >= 1.
Remember to use a capital "B".
Don't forget to return the result.
